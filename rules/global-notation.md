        # 全エージェント共通 表記レギュレーション

        このルールはすべてのエージェントに自動適用される。
        詳細なルール・固有名詞DB・禁止表記リストは以下を参照すること。

        ```
        skills/shared/regulation-notation.md
        ```

        ## 必須参照タイミング

        原稿・企画書・レビューシートを出力する前に、必ず `skills/shared/regulation-notation.md` を Read して内容を確認すること。

        ## 最重要チェック項目（抜粋）

        ### 会社名・プロダクト名

        | 表記 | 用途 |
        |---|---|
        | アトミックソフトウェア株式会社 | 正式社名（初出時） |
        | アトミックソフトウェア | 略称（日本語テキスト内） |
        | medicalforce | クリニック向けSaaS（必ず小文字始まり・一単語） |
        | 警備フォース | 警備事業者向けSaaS |

        - 「Medicalforce」「MedicalForce」「medical force」はすべて誤り
        - 「メディカルフォース」を現社名として使うことを禁止する（2026年2月6日に社名変更済み）

        ### 禁止表現（全エージェント共通）

        - 根拠なき修飾語（「画期的な」「革新的な」「唯一無二の」等）
        - 「させていただく」の過剰使用（1原稿2回まで）
        - AI特有の定型パターン（「〜と言えるでしょう」「〜ではないでしょうか」の多用）
        - 強調語（「一切」「一点集中」「完全に」等）
        - 言葉同士を「＝」でつなぐ表現
        - 並列列挙に「・」を使う表現（「や」「と」「、」で代替する）
        - 「——」を使って語句をつなぐ表現（タイトル・見出し・本文）
        - 比喩的な表現

        ### 氏名表記

        - 姓と名の間に半角スペースを入れる（例: 畠中 翔一）
        - 役職と氏名の区切りは「／」を使用（例: 代表取締役CEO／畠中 翔一）

        ### ミッション正式表記

        - 「未来に、もっと余白を。」（句点「。」まで含めて正式なミッション文）
