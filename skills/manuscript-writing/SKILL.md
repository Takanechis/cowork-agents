# 原稿ライティング共通スキル

このファイルは全カテゴリ（インタビュー・メール・チャット）に共通するライティングルールを定義する。
カテゴリ固有のルールは categories/ ディレクトリの各ファイルを参照すること。

## 企業情報

- 会社名: アトミックソフトウェア株式会社（旧：メディカルフォース株式会社）
- ブランド名: medicalforce（クリニック向けSaaS）、警備フォース（警備事業者向けSaaS）
- ビジョン: 「これからの産業の成長プロセスを合理化する」
- 戦略: マルチバーティカル戦略 — SaaSを起点としたエコシステム形成を複数産業に展開
- 思想: アトミックソフトウェア思想 — 機能をモジュール化し、組み合わせて効率的にソリューションを構築
- 公式note: https://note.com/medicalforce

## 基本トーン

- 知的で落ち着いた文体を基調とする
- 専門用語は適度に使いつつ、一般読者にもわかりやすく噛み砕く
- 過度にカジュアルにならず、かといって堅すぎず「信頼感のある親しみやすさ」を維持
- 主張に対しては必ず根拠や具体例を添える

## 表記ルール

- 会社名: 「アトミックソフトウェア」または「メディカルフォース」（文脈に応じて使い分け）
- プロダクト名: 「medicalforce」（小文字始まり、一単語）
- 数字: 算用数字を使用（一、二ではなく1、2）。ただし慣用句は漢数字OK（一つ、二つ）
- 敬称: 社内メンバーは敬称なし。社外は「様」「さん」を適宜使用
- 括弧: 強調は「」、引用・固有名詞は『』、補足は（）

## 禁止表現（全カテゴリ共通）

以下の表現は使用禁止:
- 根拠のない修飾語（「画期的な」「革新的な」「唯一無二の」等）
- 「させていただく」の過剰使用（1記事2回まで）
- 誇大広告に該当しうる表現（医療広告ガイドライン準拠）
- AI特有の定型パターン（「〜と言えるでしょう」「〜ではないでしょうか」の多用）
- 根拠なき断定（「間違いなく」「必ず」「絶対に」等）
- 過度な体言止めの連発

## 品質チェックリスト（全カテゴリ共通）

生成後に必ず以下を確認:
- [ ] 禁止表現が含まれていないか
- [ ] 事実関係がソースと矛盾しないか
- [ ] トーンが一貫しているか
- [ ] 誤字脱字がないか
- [ ] 固有名詞（人名・製品名・社名）の表記が正確か
- [ ] 文字数が媒体要件を満たしているか
