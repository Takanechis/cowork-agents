# チャットメッセージカテゴリルール

Slack・Teams等のビジネスチャットでのメッセージ作成ルール。

> **ステータス: プレースホルダー**
> 実際のチャットサンプルを学習させた後、トンマナ・テンプレートを追加予定。

---

## 1. 基本トーン

- 簡潔かつフレンドリー
- メールほど堅くなく、カジュアルすぎない
- 要件は最初の1〜2行で伝える
- 長文になる場合はスレッドを活用する前提で書く

## 2. 構成パターン

### 依頼・相談
```
@相手 お疲れ様です！
○○の件でご相談です。

[要件を1〜2文で]

[補足があれば箇条書き]

お手すきの際にご確認いただけますか？
```

### 報告・共有
```
○○の件、共有します。

[結論を1文で]
[詳細があれば箇条書き]

何かあればお知らせください！
```

### お礼・リアクション
```
ありがとうございます！
[具体的に何に対するお礼かを1文で]
```

## 3. チャット固有のルール

- 改行を活用し、視認性を高める
- 箇条書きを積極的に使う
- 絵文字は控えめに（ビジネス文脈では1〜2個まで）
- メンション（@）は適切に使う
- 長いURLはリンクテキストにする
- コードや設定値はバッククォートで囲む

## 4. 禁止表現

- 過度なカジュアル表現（「まじで」「やばい」等）
- 全角の「！」「？」の連続使用（「！！！」等）
- 主語・目的語の省略しすぎ（何の話かわからなくなる）
