## レビュー手順

### STEP 1: 機密・非開示情報の検出

原稿全文を読み、以下のキーワードが含まれていないかチェックする:

**機密キーワード（含まれていたら即アラート）:**
- 売上、ARR、CAR、MRR、収益、利益、粗利
- 給与、年収、報酬、賞与
- 資金調達、バリュエーション、時価総額
- 株主総会、取締役会、議事録
- M&A、買収、合併
- 財務諸表、貸借対照表、損益計算書
- 事業計画、予算、投資計画

**非開示キーワード（含まれていたら確認）:**
- 定款、登記簿、印鑑証明
- 社内規程、就業規則
- ローンチ予定、リリース予定（未公開のもの）
- 人事異動（社内周知前）

### STEP 2: 公開情報の正確性チェック

原稿に含まれる企業情報（数字・名称）が公開情報リファレンスと一致するか確認する:

- [ ] 会社名・英語表記が正しいか
- [ ] 住所・所在地が最新か
- [ ] 資本金の表記が推奨フォーマットか
- [ ] 役員名・肩書きが最新か
- [ ] 従業員数・導入院数が最新の数字か
- [ ] ボイラープレートが最新版か

### STEP 3: レビュー結果の記録

| チェック項目 | 結果 | 備考 |
|-----------|------|------|
| 機密情報の混入なし | ○ / × | |
| 非開示情報の混入なし | ○ / × | |
| 公開情報の数値が最新 | ○ / × | |
| ボイラープレートが最新版 | ○ / × | |
| 役員情報が最新 | ○ / × | |

## アラート発報ルール

原稿に機密情報または非開示情報が検出された場合:

1. **即座に該当箇所をハイライトする**
2. **以下のフォーマットでアラートを出力する:**

```
⚠️ 情報開示アラート
━━━━━━━━━━━━━━━━━━━━
検出レベル: 🔴 機密 / 🚫 非開示
該当箇所: 「（原稿の該当テキスト）」
情報分類: （ジャンル > 大項目 > 情報名）
対応: 削除必須 / 担当部門に確認必要
━━━━━━━━━━━━━━━━━━━━
```

3. **機密情報の場合: 該当箇所を削除し、代替表現を提案する**
4. **非開示情報の場合: 該当箇所を伏せ、担当部門への確認を促す**
