    # 校正スキル: プレスリリース

    ## Overview

    参照ライティングスキル: skills/writing/press-release.md
    目的: 配信前の最終品質確認。表記・構成・法的リスクをチェックする

    ---

    ## Phase 1: 表記チェック

    - 社名・プロダクト名・役職者氏名の正確な表記
    - regulation-notation.md の禁止表現の有無
    - 感嘆語・装飾語の有無
    - タイポ・誤変換・表記ゆれ
    - 推測・不明な数字に「要確認」ラベルがあるか

    ---

    ## Phase 2: 構成・法的チェック

    - タイトルに数字または固有名詞が含まれているか
    - リードパラグラフが 3文以内か
    - 会社概要・問い合わせ先が的確か
    - 誤大広告に該当する表現がないか
    - 医療広告ガイドラインに違反する表現がないか
    - 入力情報と数字・固有名詞の整合性がとれているか

    ---

    ## 指摘出力フォーマット

    ```
    校正レポート（{件数}件）
    {番号}. {指摘内容}
      該当箇所: {引用}
        修正案: {具体的なテキスト}
        ```

        ---

        ## Output Rules

        - 出力は Markdown で行う（** の使用は禁止）
        - 指摘は「該当箇所の引用」と「修正案」を必ずセットで示す
        - 問題なし・OK の事項はアウトプットに含めない
        - 褒める表現は使用しない
        - 言葉のトンマナは「ですます調」に統一する
