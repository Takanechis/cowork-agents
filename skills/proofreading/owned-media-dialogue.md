    # 校正スキル: オウンドメディア 対談記事

    ## Overview

    参照ライティングスキル: skills/writing/owned-media-dialogue.md
    目的: 投稿前の最終品質確認。表記・構成・内容を多角的にチェックする

    ---

    ## Phase 1: 表記チェック

    - 2者の氏名・所属・役職の正確な表記
    - regulation-notation.md の禁止表現の有無
    - AIらしい語彙・定型表現の有無
    - タイポ・誤変換・表記ゆれ
    - 話者名の表記形式の一貫性

    ---

    ## Phase 2: 構成・品質チェック

    - タイトルに固有名詞または数字が含まれているか
    - リード文が 3段落構成か
    - 中見出しが 3〜5個配置されているか
    - 話者名が「{話者名}：」の形式で一貫しているか
    - 2者の視点の交差・発展が本文で示されているか
    - 单なる同意の連続になっていないか
    - 取材・文クレジットが末尾にあるか

    ---

    ## 指摘出力フォーマット

    ```
    校正レポート（{件数}件）

    {番号}. {指摘内容}
      該当箇所: {引用}
        修正案: {具体的なテキスト}
        ```

        ---

        ## Output Rules

        - 出力は Markdown で行う（** の使用は禁止）
        - 指摘は「該当箇所の引用」と「修正案」を必ずセットで示す
        - 問題なし・OK の事項はアウトプットに含めない
        - 褒める表現は使用しない
        - 言葉のトンマナは「ですます調」に統一する
