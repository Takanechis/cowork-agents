    # 企画スキル: オウンドメディア インタビュー記事

    ## Overview

    対象媒体: note（社内インタビュー）・VerticalSaaS Mag.（外部取材）
    目的: 取材目的・読者層・骨子を定義し、画質の高い記事の型を作る

    ---

    ## 出力フォーマット

    ### 企画タイトル案（2〜3パターン）

    - 引用フレーズ（話者の言葉から抜粋）＋サブテーマの二部構成
    - 固有名詞・数字・具体的な問いを含む

    ### インタビュイ情報

    - 氏名・役職・経歴
    - 担当領域・チーム構成

    ### 企画骨子（テーマ1〜3 + 各深掘りポイント）

    - テーマ1: 話者の実体験・思考プロセスの核心
    - テーマ2: 読者が持ち帰れる知見
    - テーマ3: 今後の展望・提言

    ### メモ: 企画概要（200〜300字）

    ---

    ## リサーチ手順（外部取材の場合）

    1. 公式サイト — 事業内容・ミッション・沿革・経営陣
    2. note / ブログ — 創業ストーリー・失敗談
    3. PR TIMES — 資金調達・新サービス・提携情報
    4. X (Twitter) — 本人の発信・業界内での評判
    5. メディア記事 — 既存取材の切り口を把握

    ---

    ## 品質ゲート

    - タイトル案が 2〜3パターン生成されている
    - テーマが 3つあり、各テーマに深掘りポイントが 3点以上ある
    - 企画概要が 200〜300字程度ある
    - 禁止表現が 0件
