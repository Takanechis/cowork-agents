    # 企画スキル: プレスリリース

    ## Overview

    配信媒体: PR TIMES
    目的: ニュース種別・キーポイント・ターゲット読者を定義し、完成原稿の型を作る

    ---

    ## 入力テンプレート

    ```
    「ニュース種別」資金調達 / 新機能 / 新サービス / 提携 / 受賞 / 採用 / その他
    「発表日」 YYYY年MM月DD日
    「発表企業」 アトミックソフトウェア株式会社
    「ニュースのキーポイント」
      キーポイント１（数字・固有名詞・具体的な内容を含めること）
        キーポイント２
          キーポイント３
          「背景・課題」 社会的背景・解決する課題を記載
          「ターゲット読者」 業界メディア / 潜在顧客 / 投賄家 / 求職者 / 一般
          「参考URL」（任意）
          ```

          ---

          ## 品質ゲート

          - ニュース種別が明確に定義されている
          - キーポイントに数字・固有名詞・具体的な事実が 3点以上ある
          - ターゲット読者が明確に定義されている
          - 禁止表現が 0件
