# ペルソナ: PR・広報担当者

## 基本情報

- 名称: PR・広報担当者ペルソナ
- 職種・役割: 企業の広報・PR担当者、またはオウンドメディア（note）の主要読者
- 想定読者層: スタートアップ・SaaS企業のPR担当、社内広報チーム、採用広報に関わるメンバー
- 媒体: アトミックソフトウェア公式note（社員インタビュー記事）

## 知識レベル

- SaaS業界の基礎知識: 中〜高 — 「SaaS」「バーティカルSaaS」程度は説明不要。ただし「medicalforce」「警備フォース」など個別プロダクトの詳細は初見の可能性がある
- 医療・クリニック業界の専門知識: 低〜中 — 「電子カルテ」「予約システム」程度は理解できるが、臨床用語や医療保険の詳細は噛み砕いた説明が必要
- 採用・カルチャー: 高 — 「コアバリュー」「ミッション」「エンプロイヤーブランディング」は前提知識として持っている
- スタートアップ・経営: 中 — 「IPO」「シリーズ」「バーン」などスタートアップ用語はわかる。複雑な財務・会計用語は噛み砕く

## 関心事・優先事項

- 候補者に「この会社で働きたい」と思わせるリアルな社員の姿が伝わること
- 社員の言葉や体験談が具体的で、共感を生む表現になっていること
- アトミックソフトウェアの思想・カルチャー（コアバリュー、ミッション）が自然に伝わること
- 表現がファクトに基づいており、誇張や誤りがないこと
- 記事として公開しても違和感のない文体・構成になっていること

## 受け取りやすい言い回し・補足粒度

- 「medicalforce」を初出に使う場合は、「クリニック向けオールインワンSaaS『medicalforce』」のように一言説明を添えると親切
- 「バーティカルSaaS」は読者が知っている前提でよいが、初出時に「特定業界に特化したSaaS（バーティカルSaaS）」と補足してもよい
- 社員の発言に業界専門用語が含まれる場合、括弧内や直後の文で一言解説を加える
- ミッション「未来に、もっと余白を。」はそのまま引用してよい（句点まで含めて引用符で囲む）
- コアバリュー（HRT / We / 感動 / Punk）は略称のままでよいが、初出時に「HRT（Humility・Respect・Trust）」と補足すると親切
- 数字・実績（導入院数700院以上、継続利用率99.2%など）はそのまま使ってよい
- 「です・ます調」の統一を前提に読んでいる

## 避けるべき表現・前提

- 「革新的な」「画期的な」などの根拠なき修飾語は読者の信頼を損なう
- 「させていただく」の過剰使用は不自然に感じる
- 「AIらしい」定型文（「〜と言えるでしょう」「〜に貢献しています」の多用）は読者が拒否反応を示す
- 医療・法令・財務の具体的な数字や規制に関する言及は、出典や文脈なしに書かない
- 廃止されたビジョン「産業の成長プロセスを合理化する」は使用禁止

## レビューエージェントへの指示

このペルソナが読む原稿をレビューする際、以下を確認する。

1. 専門用語の補足: SaaSや業界固有の用語が初出で説明なしに使われていないか
2. 具体性: 社員の発言や体験談が「どんな業務で」「どう感じたか」が伝わる具体度になっているか
3. カルチャー伝達: アトミックソフトウェアの思想・コアバリューが自然な流れで伝わっているか
4. ファクト確認: 数字・役職・社名・プロダクト名の表記がregulation-notation.mdと一致しているか
5. 共感ポイント: 読者（転職検討者、採用担当者）が「この会社で働きたい」と感じる表現になっているか
