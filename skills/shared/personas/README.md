# personas/ — ターゲットペルソナ定義

このフォルダは、各エージェントが原稿・成果物をレビュー・生成する際に参照するターゲットペルソナを定義する。

## 目的

- 誰に読まれるコンテンツかを明示し、エージェントが読者に合った補足・言い回しを選択できるようにする
- ペルソナの知識レベル・関心事・文脈に応じて、専門用語の説明粒度や具体例の選び方を調整する

## フォルダ構成

personas/
  README.md                    # このファイル（ペルソナフォルダの概要）
  persona-pr-manager.md        # ペルソナ: PR・広報担当者
  persona-media-journalist.md  # ペルソナ: メディア記者・ライター

## ペルソナの使い方（エージェント向け）

レビューエージェントは、以下の手順でペルソナを活用する。

1. 原稿・成果物の種別（インタビュー記事 / プレスリリース / メール 等）から読者ペルソナを特定する
2. 該当するペルソナファイルを読み込む
3. ペルソナの「知識レベル」「関心事」「受け取りやすい言い回し」を踏まえてレビュー・補足を行う

## ペルソナと原稿種別の対応表

| 原稿種別 | 主なターゲットペルソナ |
|---|---|
| インタビュー記事（オウンドメディアnote） | persona-pr-manager.md |
| プレスリリース | persona-media-journalist.md |
| メール（社外：メディア宛） | persona-media-journalist.md |
| メール（社外：クライアント・パートナー宛） | persona-pr-manager.md |

## ペルソナの追加・更新フロー

1. 新しいペルソナが必要になったら persona-{識別子}.md を作成する
2. ペルソナテンプレート（後述）に従って記述する
3. この README の対応表を更新する
4. 該当エージェントの description または Workflow に参照先を追記する
